# Shortest-Route-Finder
# 🚗 City Navigation System

A C++ based City Navigation System that demonstrates the practical implementation of Graph Data Structures and fundamental graph algorithms such as BFS, DFS, and Dijkstra's Algorithm.

## 📌 Overview

This project models cities as nodes and routes as edges in a graph. Users can create connections between cities, explore routes using graph traversal algorithms, and find the shortest path between locations.

The project was developed to strengthen understanding of graph theory concepts and their real-world applications in navigation systems.

## ✨ Features

* Add cities and routes
* Represent transportation networks using graphs
* Breadth First Search (BFS) Traversal
* Depth First Search (DFS) Traversal
* Shortest Path Calculation using Dijkstra's Algorithm
* User-friendly console interface
* Dynamic route management

## 🛠 Technologies Used

* C++
* STL (Standard Template Library)
* Graph Data Structures
* BFS Algorithm
* DFS Algorithm
* Dijkstra's Algorithm

## 🚀 How to Run

### Compile

```bash
g++ main.cpp -o navigation
```

### Execute

```bash
./navigation
```
